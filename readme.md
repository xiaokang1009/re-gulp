# Gulp项目构建工具

使用Gulp构建项目，包括以下功能：
 - 自动替换html中的引用文件路径
 - 自动编译sass文件
 - 自动合成雪碧图
 - 自动将utf8编码的文件转换为gbk编码
 - 支持css、js、html文件的压缩
 - 自动转换es6语法

## 使用方法
进入项目目录，执行以下命令：
```bash
  npm install or yarn install or pnpm install # 安装依赖
  ## 全局安装re-gulp
  npm run build:link # 打包，并安装re-gulp
```
创建模板项目：
```bash
  create-re-gulp <projectName> # 创建模板项目
```
